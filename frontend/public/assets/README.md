# 美术资源目录

## 目录结构

```
assets/
├── icons/              # 图标资源
│   ├── actions/        # 动作图标 (2PM, 3PM, AST等)
│   ├── controls/       # 控制图标 (播放、暂停等)
│   └── ui/             # UI图标 (设置、统计等)
├── images/             # 图片资源
│   ├── jerseys/        # 球衣图标
│   ├── court/          # 篮球场背景
│   └── avatars/        # 球员头像占位符
└── fonts/              # 字体文件（如需要）
```

## 资源说明

所有资源使用占位符，后期可以替换为专业设计的美术资源。
